ERP de gestion d'IUT (En cours...)

Lors de ce projet, une classe entière de 20 personnes a dû s'organiser et se coordonner afin de créer une application web dont l'objectif serait de gérer le département Informatique de l'IUT.

Pour ce projet, nous avons fait le choix d'utiliser Angular afin de réaliser le Front-End et NestJS pour l'API REST.
